# Tabel Pengujian Black Box

Berikut adalah tabel pengujian Black Box untuk sistem Informasi Pendaftaran Peserta Didik Baru (SIPPDB) SMK. Pengujian ini mencakup seluruh fungsionalitas sistem mulai dari akses publik, pendaftaran siswa, hingga manajemen admin.

|  No   | Skenario Pengujian                        | Test Case                                        | Hasil yang Diharapkan                                                                                                          | Hasil Pengujian                             | Kesimpulan |
| :---: | :---------------------------------------- | :----------------------------------------------- | :----------------------------------------------------------------------------------------------------------------------------- | :------------------------------------------ | :--------: |
| **1** | **Akses Halaman Utama (Public)**          |                                                  |                                                                                                                                |                                             |            |
|  1.1  | Mengakses Halaman Beranda                 | [Gambar Tampilan Halaman Beranda]                | Halaman utama dapat diakses dan menampilkan informasi sekilas tentang sekolah serta banner ucapan selamat datang.              | [Gambar Hasil Screenshot Halaman Beranda]   |  Berhasil  |
|  1.2  | Mengakses Halaman Profil Sekolah          | [Gambar Menu Profil Sekolah]                     | Halaman menampilkan profil, visi, misi, dan sejarah singkat sekolah.                                                           | [Gambar Hasil Screenshot Profil Sekolah]    |  Berhasil  |
|  1.3  | Mengakses Halaman Panduan PPDB            | [Gambar Menu Panduan]                            | Halaman menampilkan langkah-langkah atau alur pendaftaran PPDB dengan jelas.                                                   | [Gambar Hasil Screenshot Panduan PPDB]      |  Berhasil  |
|  1.4  | Mengakses Halaman Jadwal Kegiatan         | [Gambar Menu Jadwal]                             | Halaman menampilkan tabel jadwal gelombang pendaftaran yang aktif maupun yang akan datang.                                     | [Gambar Hasil Screenshot Jadwal PPDB]       |  Berhasil  |
|  1.5  | Mengakses Halaman Pengumuman              | [Gambar Menu Pengumuman]                         | Halaman menampilkan daftar pengumuman terbaru terkait PPDB.                                                                    | [Gambar Hasil Screenshot Daftar Pengumuman] |  Berhasil  |
| **2** | **Registrasi dan Autentikasi**            |                                                  |                                                                                                                                |                                             |            |
|  2.1  | Pendaftaran Akun Baru (Register)          | [Gambar Form Pendaftaran Akun Kosong]            | Form pendaftaran tampil. Setelah data diisi dengan valid dan disubmit, sistem membuat akun dan mengarahkan ke dashboard/login. | [Gambar Form Pendaftaran Akun Terisi]       |  Berhasil  |
|  2.2  | Login User (Calon Siswa)                  | [Gambar Form Login dengan Data Valid]            | Sistem memverifikasi kredensial dan mengarahkan pengguna ke Dashboard Calon Siswa.                                             | [Gambar Dashboard Utama Siswa]              |  Berhasil  |
|  2.3  | Login Admin                               | [Gambar Form Login dengan Akun Admin]            | Sistem memverifikasi kredensial admin dan mengarahkan ke Dashboard Admin.                                                      | [Gambar Dashboard Utama Admin]              |  Berhasil  |
| **3** | **Proses Pendaftaran (Calon Siswa)**      |                                                  |                                                                                                                                |                                             |            |
|  3.1  | Mengakses Menu Pendaftaran                | [Gambar Dashboard - Menu Pendaftaran]            | Pengguna diarahkan ke halaman formulir pendaftaran tahap awal.                                                                 | [Gambar Halaman Index Pendaftaran]          |  Berhasil  |
|  3.2  | Mengisi Formulir Tahap 1 (Data Pribadi)   | [Gambar Form Data Pribadi Terisi Lengkap]        | Data tersimpan dan pengguna lanjut ke tahap berikutnya (Data Orang Tua).                                                       | [Gambar Form Step 1 - Data Pribadi]         |  Berhasil  |
|  3.3  | Mengisi Formulir Tahap 2 (Data Orang Tua) | [Gambar Form Data Orang Tua Terisi Lengkap]      | Data orang tua tersimpan dan pengguna lanjut ke tahap berikutnya (Upload Berkas).                                              | [Gambar Form Step 2 - Data Orang Tua]       |  Berhasil  |
|  3.4  | Mengisi Formulir Tahap 3 (Upload Berkas)  | [Gambar Form Upload Berkas dengan File Terpilih] | Berkas berhasil diunggah (KK, Akta, Ijazah/SKL) dan pengguna lanjut ke konfirmasi.                                             | [Gambar Form Step 3 - Upload Berkas]        |  Berhasil  |
|  3.5  | Finalisasi Pendaftaran (Konfirmasi)       | [Gambar Halaman Konfirmasi Data]                 | Sistem menampilkan rangkuman data. Setelah konfirmasi, status pendaftaran berubah menjadi "Menunggu Verifikasi".               | [Gambar Halaman Step 4 - Konfirmasi]        |  Berhasil  |
| **4** | **Dashboard & Fitur Siswa**               |                                                  |                                                                                                                                |                                             |            |
|  4.1  | Melihat Status Pendaftaran                | [Gambar Dashboard Siswa Status Pendaftaran]      | Dashboard menampilkan status terkini (Menunggu Verifikasi, Diterima, atau Ditolak).                                            | [Gambar Dashboard Utama Siswa]              |  Berhasil  |
|  4.2  | Melihat Pengumuman Kelulusan              | [Gambar Menu Pengumuman di Dashboard]            | Jika sudah ada pengumuman, siswa dapat melihat status kelulusan mereka.                                                        | [Gambar Halaman Pengumuman Siswa]           |  Berhasil  |
|  4.3  | Kelola Berkas / Perbaikan Berkas          | [Gambar Halaman Kelola Berkas]                   | Siswa dapat melihat berkas yang diunggah dan memperbaikinya jika status belum terkunci/diminta perbaikan.                      | [Gambar Halaman Kelola Berkas]              |  Berhasil  |
|  4.4  | Cetak Bukti Pendaftaran                   | [Gambar Tombol Cetak Bukti]                      | Sistem men-generate file PDF bukti pendaftaran yang berisi data siswa.                                                         | [Gambar Halaman Cetak Bukti Pendaftaran]    |  Berhasil  |
| **5** | **Manajemen Admin**                       |                                                  |                                                                                                                                |                                             |            |
|  5.1  | Dashboard Statistik Admin                 | [Gambar Dashboard Admin]                         | Menampilkan statistik jumlah pendaftar, jumlah jurusan, dan status verifikasi.                                                 | [Gambar Dashboard Utama Admin]              |  Berhasil  |
|  5.2  | Kelola Jurusan (Tambah/Lihat)             | [Gambar Halaman Kelola Jurusan]                  | Admin dapat melihat daftar jurusan dan menambahkan jurusan baru.                                                               | [Gambar Halaman Daftar Jurusan]             |  Berhasil  |
|  5.3  | Kelola Gelombang Pendaftaran              | [Gambar Halaman Kelola Gelombang]                | Admin dapat mengatur gelombang pendaftaran (buka/tutup, tanggal mulai/selesai).                                                | [Gambar Halaman Daftar Gelombang]           |  Berhasil  |
|  5.4  | Kelola Data Pengguna (Users)              | [Gambar Halaman Kelola User]                     | Admin dapat melihat daftar pengguna terdaftar dan mengelola akses mereka.                                                      | [Gambar Halaman Daftar User]                |  Berhasil  |
|  5.5  | Kelola Data Calon Siswa                   | [Gambar Halaman Data Calon Siswa]                | Admin dapat melihat seluruh data calon siswa yang mendaftar.                                                                   | [Gambar Halaman Daftar Calon Siswa]         |  Berhasil  |
| **6** | **Proses Verifikasi (Admin)**             |                                                  |                                                                                                                                |                                             |            |

|  6.2  | Validasi Data Pendaftar                   | [Gambar Halaman Detail Pendaftar]                | Admin memvalidasi kesesuaian data inputan dengan berkas fisik/digital.                                                         | [Gambar Halaman Detail Calon Siswa]         |  Berhasil  |

|  7.2  | Export Data ke Excel/PDF                  | [Gambar Tombol Export Laporan]                   | Sistem mengunduh file laporan dalam format Excel/PDF sesuai filter yang dipilih.                                               | [Gambar Output File Export]                 |  Berhasil  |
